# save-juejin-booklet

Easily save the Juejin booklet.

## usage

1. Install Package

```shell
npm i
```

2. Save Cookies

```shell
npm run save:cookies
```

Then, login with WeChat.

3. Save Booklet

```shell
npm run save:booklet
```
